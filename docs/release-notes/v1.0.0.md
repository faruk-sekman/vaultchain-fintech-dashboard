# Vaultchain v1.0.0

Vaultchain 1.0.0 is the first public portfolio release of the fintech operations console.

Highlights:

- Angular 21 console and NestJS 11 API over PostgreSQL 16.
- MFA-aware authentication, RBAC, masked PII, KYC workflow, and audited operator actions.
- Integer minor-unit double-entry ledger with idempotent writes.
- Read-only, non-custodial Web3 risk view with clearly labelled simulated signals.
- Docker demo, committed OpenAPI contract, unit/integration/E2E test lanes, and blocking security gates.

This release uses synthetic seed data only. It is a case-study application, not a production banking
service or a hosted custody product. The next visible work is tracked in the [roadmap](../roadmap.md)
and the ready-to-publish [roadmap issue drafts](../roadmap-issues.md).
